码农笔记

为了防止电脑系统崩溃了之后文件全部丢失，整理备份下电脑里安装过的软件及其下载路径，电脑里的文件和文件的来源用途

软件

日常

  • 微信
  • TIM
  • 网易云
  • WPS
  • WinRAR
  • 电脑管家
  • Firefox
  • Chrome
  • 百度网盘
  • 迅雷
  • 搜狗输入法

IDEA（集成开发环境）

  • visual studio
  • visual studio code
  • Android studio
  • eclipse for Java
  • eclipse for PHP
  • 微信web开发者工具
  • Java运行环境
  • IDEA

网络安全工具

  • nmap（网络扫描和嗅探工具包）
  • xshell（远程登录工具）

计算机类

  • VMware（虚拟机）
  • Git（版本控制工具）
  • GitHub desktop
  • mindmanager
  • node.js（hexo博客的运行环境）
  • Navicat（数据库管理工具）

文件

日常文件

  • wstmart（B2B2C商城文件）
  • NUAA图书馆电子书爬虫
  • 书籍类

系统

  • centos
  • Ubuntu
  • Ubuntu server
  • kali
  • Debian
  • nethunter（安卓版kali）
